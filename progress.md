## Что уже понимаю

- Переменные: имя, значение, присваивание через `=`
- Разница между строкой (`"31"`) и числом (`31`)
- `print()` — вывод в терминал; запятая между аргументами добавляет пробел
- Склейка строк через `+` — контроль пробелов и знаков препинания
- `input()` — ввод от пользователя, **всегда** возвращает строку
- `int()` — превращает строку в целое число для арифметики
- Арифметика: `+`, `-`, `*`, `/` (деление может дать дробь)
- `"10" + "3"` → `"103"` (склейка), `10 + 3` → `13` (сложение)
- Запуск: сохранить файл (Ctrl+S) → `python lessons/day_XX.py`
- `if` / `else` — ветвление; сравнения: `>`, `<`, `>=`, `<=`, `==`, `!=`
- `=` — присваивание, `==` — сравнение «равно»
- После `if` / `else` — двоеточие `:` и отступ у кода внутри блока
- Цикл `for` — повторение блока кода; `for переменная in range(...):`
- `range(начало, конец)` — конец **не входит**; до N включительно → `range(1, N + 1)`
- Накопление в цикле: переменная до цикла (`z = 0`), в цикле `z = z + i`
- `_` в `for _ in range(n)` — когда номер шага не нужен, важен только счёт повторов
- `if` внутри `for` — условие на каждом шаге цикла; вложенные отступы
- `%` — остаток от деления; `i % 2 == 0` — число чётное
- Накопление: произведение — старт `1`, в цикле `product = product * i`
- `if` / `elif` / `else` — выбор **одной** ветки из нескольких
- `and` — оба условия истинны (`op == "/" and b == 0`)
- Мини-проект: калькулятор с проверкой деления на ноль
- Цикл `while` — повтор, **пока** условие истинно; менять переменную в условии
- `or` — хотя бы одно условие истинно (`age < 1 or age > 120`)
- `not` — отрицание условия
- Зеркало: «оба верны» (`and` + `==`) ↔ «крутить while, пока хоть одно неверно» (`or` + `!=`)
- `def` — именованный блок кода; параметры передают данные внутрь
- `return` — отдать значение вызывающему коду; ловить в переменную: `e = func()`
- `print` в функции ≠ `return` — для `if` и расчётов нужен `return`
- `len()`, `.strip()`, `.lower()` — работа с текстом из input
- `!=` + **or** → «хоть одно неверно»; `==` + **and** → «оба верны»

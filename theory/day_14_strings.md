# День 14 — строки: методы и работа с текстом

## Зачем это придумали

**Проблема:** почти всё, что приходит **извне** — текст: ввод пользователя, сообщение в чате, строка из файла, ответ API. `"25"` — строка, не число; `"Да"` и `"да"` — **разные** строки для Python. Без работы со строками программы ломаются на «мелочах».

**Зачем методы строк:** не писать десять `if` на каждый регистр и пробел. Язык даёт готовые операции: «сделай маленькими буквами», «убери пробелы», «сколько символов» — одной командой.

**В ИИ-проектах** строки — 90% входа и вывода: промпт, ответ модели, имя файла, кнопка «отправить».

## Строка — последовательность символов

```python
word = "Python"
print(word[0])   # P — первый символ (индекс 0)
print(len(word)) # 6 — длина
```

**`len(строка)`** — сколько символов (пробелы и знаки тоже считаются).

---

## Пробелы и регистр

Пользователь часто вводит лишнее:

```python
answer = "  Да  "
print(answer)        # с пробелами
print(answer.strip())  # "Да" — пробелы по краям убраны
```

```python
cmd = "STOP"
print(cmd.lower())   # "stop"
print(cmd.upper())   # "STOP"
```

**`.strip()`** — убрать пробелы слева и справа.  
**`.lower()`** / **`.upper()`** — все буквы маленькие / большие.

Сравнение без сюрпризов:

```python
if answer.strip().lower() == "да":
    print("Ок")
```

## Проверка «есть ли внутри»

```python
email = "user@mail.com"
if "@" in email:
    print("Похоже на email")
```

**`in`** — «содержится ли подстрока».

---

## Перебор символов строки

Строку можно пройти циклом `for` — **по символам**:

```python
for ch in "hello":
    print(ch)
```

Так считают буквы, ищут ошибки, проверяют каждый символ.

---

## Строки не меняются «на месте»

```python
name = "Сергей"
name.lower()   # вернёт "сергей", но name всё ещё "Сергей"
```

Методы **возвращают новую** строку. Чтобы сохранить:

```python
name = name.lower()
```

Как с `return` — результат нужно **куда-то положить**.

## Частые ошибки

| Ошибка | Что происходит |
|--------|----------------|
| `"да" == "Да"` | False — разный регистр |
| забыли `.strip()` | `" да "` не равно `"да"` |
| `len > 0` vs пустая после strip | `"   "` кажется непустой |
| думали, что `.lower()` меняет переменную сама | старое значение остаётся |

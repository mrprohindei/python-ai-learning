# Теория курса

Здесь — развёрнутые объяснения к каждому дню.  
Читать **до** задач в `lessons/day_XX.py`.

| День | Файл | Тема |
|------|------|------|
| 8 | [day_08_for.md](day_08_for.md) | цикл `for`, `range` |
| 9 | [day_09_for_if.md](day_09_for_if.md) | `for` + `if`, оператор `%` |
| 10 | [day_10_calculator.md](day_10_calculator.md) | мини-калькулятор, `elif` |
| 11 | [day_11_while.md](day_11_while.md) | цикл `while` |
| 12 | [day_12_and_or_not.md](day_12_and_or_not.md) | `and`, `or`, `not` |
| 13 | [day_13_functions.md](day_13_functions.md) | функции `def`, `return` |
| 14 | [day_14_strings.md](day_14_strings.md) | строки: `len`, `.lower()`, `.strip()` |
| 15 | [day_15_quiz.md](day_15_quiz.md) | мини-викторина (сборка) |

Файлы добавляются по мере прохождения курса. В чате — краткое резюме + ответы на вопросы.

В каждой теории есть блок **«зачем придумали»** — какую проблему решает тема в программировании вообще, не только синтаксис.

### `range(конец)` не включает последнее число

- **Тема:** День 9, `for` + `range`
- **Какой код был написан:** `for i in range(1, 20):` — нужно было дойти до 20 включительно
- **Какая ошибка возникла:** вывелись чётные 2…18, а 20 пропало
- **Почему она возникла:** `range(a, b)` идёт до `b - 1`, конец **не входит**
- **Как исправить:** `range(2, 21)` или `range(1, 20 + 1)`
- **Как запомнить на будущее:** нужна N включительно → пиши `N + 1` в конце `range`
- **Тренировка:** выведи числа 1…10 так, чтобы 10 точно попало. Проверь последнюю строку

### `and` вместо `or` в условии цикла

- **Тема:** День 12, логин/пароль (задача B), `while`
- **Какой код был написан:** `while n != "admin" and x != "123":`
- **Какая ошибка возникла:** цикл выходил, даже если верно только одно из двух → «Доступ разрешён» при неверном пароле
- **Почему она возникла:** `and` = «оба неверны». Нужно «повторять, пока **хоть одно** неверно»
- **Как исправить:** `while n != "admin" or x != "123":`
- **Как запомнить на будущее:** «пустить, когда **оба** верны» → `==` + `and`; «не пустить, когда **что-то** не так» → `!=` + `or`
- **Тренировка:** на бумаге заполни таблицу 4 строк (верно/неверно × верно/неверно): когда цикл крутится с `or`, когда с `and`

---

### Снова `and`/`or` — теперь в `if` (та же путаница)

- **Тема:** День 14D, проверка логина и пароля
- **Какой код был написан:** `if f != "user" and z != "12345":` для отказа во входе
- **Какая ошибка возникла:** отказ только когда **оба** неверны; при одном верном пускало
- **Почему она возникла:** «отказать, если хоть что-то не так» = `or`, а не `and`
- **Как исправить:** `if f != "user" or z != "12345":`
- **Как запомнить на будущее:** в голове звучало правильно («или») — но рука написала `and`. Перед запуском **проверять логику словами**: «когда именно должно сработать?»
- **Тренировка:** напиши проверку «пускать, только если возраст >= 18 И есть билет» — каким оператором?

### `return print(...)` вместо `return значение`

- **Тема:** День 13, функция `ticket_price`
- **Какой код был написан:** `return print("Стоимость - 0")` внутри функции
- **Какая ошибка возникла:** функция отдавала `None` вместо числа; снаружи нечего было сравнивать/печатать как цену
- **Почему она возникла:** `print` только показывает текст, сам возвращает `None`. `return` должен отдавать **значение** (число/строку/True-False)
- **Как исправить:** `return 0` (число), а печатать — снаружи: `print("Стоимость:", цена)`
- **Как запомнить на будущее:** функция **считает и возвращает** (`return`), а **показывает** пусть код снаружи (`print`)
- **Тренировка:** функция `square(x)` возвращает `x*x`. Снаружи поймай в переменную и распечатай. Без `print` внутри функции

### Забыл поймать результат функции / печатал не ту переменную

- **Тема:** День 13D, регистрация
- **Какой код был написан:** долго не присваивал результат функции переменной; потом в `print` вывел старую `y` вместо обработанной `e`
- **Какая ошибка возникла:** `if` работал по одному значению, а на экран шло другое (или результат «терялся»)
- **Почему она возникла:** `return` отдаёт значение, но если не сделать `e = func(...)`, его никто не ловит. И в выводе надо брать **ту же** переменную, что в `if`
- **Как исправить:** `e = data(x, y)`, дальше и в `if`, и в `print` использовать `e`
- **Как запомнить на будущее:** цепочка `return → переменная → if/print`. Одна переменная на весь путь
- **Тренировка:** функция `clean(text)` возвращает `text.strip().lower()`. Поймай в переменную и используй её **и** в `if`, **и** в `print`

### Ошибка: `input()` возвращает строку

- **Тема:** День 2–3, `input`
- **Какой код был написан:** `age = input("Сколько лет? ")` и потом `age + 5`
- **Какая ошибка возникла:** `TypeError` — нельзя сложить строку и число; или странный результат вместо суммы
- **Почему она возникла:** `input` **всегда** отдаёт текст, даже если ввели цифры. `"25" + 5` — не то же самое, что `25 + 5`
- **Как исправить:** преобразовать ввод в число там, где нужна арифметика
- **Как запомнить на будущее:** всё из `input` — строка, пока сам не превратил в число
- **Тренировка:** спроси два числа, выведи только сумму. Проверь: `3 + 4` должно дать `7`, а не `34`

### Ошибка: забыли `int()` перед арифметикой

- **Тема:** День 3, числа
- **Какой код был написан:** `a = input("Первое число: ")` и `print(a + b)` без преобразования
- **Какая ошибка возникла:** склейка `"12" + "3"` → `"123"` вместо `15`; или `TypeError` при смешивании типов
- **Почему она возникла:** арифметика работает с числами, а в переменных лежат строки
- **Как исправить:** обернуть результат `input` в `int()` (или `float()` для дробей — позже)
- **Как запомнить на будущее:** `input` → строка → `int()` → число → можно считать
- **Тренировка:** введи `10` и `20`. Программа должна вывести `30`, не `1020`

---

### Ошибка: склейка строк и сложение чисел — разные вещи

- **Тема:** День 1, строки vs числа
- **Какой код был написан:** `age = "31"` и `age + 5`
- **Какая ошибка возникла:** `TypeError: can only concatenate str (not "int") to str`
- **Почему она возникла:** кавычки делают значение **текстом**. `"31" + 5` Python пытается «склеить», а не сложить
- **Как исправить:** убрать кавычки у числа: `age = 31`
- **Как запомнить на будущее:** кавычки = текст; без кавычек = число. `+` для текста склеивает, для чисел считает
- **Тренировка:** две переменные — одна строка `"5"`, одна число `5`. Что выведет `print(x + 3)` в каждом случае? Запиши ответ до запуска, потом проверь

### Ошибка: лишние пробелы в `print` (бонус, День 1)

- **Тема:** День 1, `print`
- **Какой код был написан:** `print("Привет", name, ", я из", city, ".")`
- **Какая ошибка возникла:** не ошибка, но вывод: `Привет Сергей , я из Норильска .` — пробелы перед `,` и `.`
- **Почему она возникла:** запятая в `print` вставляет пробел **между** аргументами; знаки препинания были отдельными кусками
- **Как исправить:** прилепить знаки к соседней строке через `+` или собрать всё в одну строку
- **Как запомнить на будущее:** `print(a, b, c)` → между a, b, c всегда пробел
- **Тренировка:** выведи `Привет, мир!` одним `print` с переменной `name = "мир"` — без пробела перед запятой и восклицательным знаком

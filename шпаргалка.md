## Переменные

```python
name = "Сергей"    # строка (текст) — в кавычках
age = 31           # число — без кавычек
city = "Норильск"
```

| Тип | Пример | Кавычки? |
|-----|--------|----------|
| строка | `"31"`, `'привет'` | да |
| число | `31`, `3.14` | нет |

**Присваивание:** `=` — кладём значение в переменную (не «равно» в математическом смысле).

## `print()` — вывод в терминал

```python
print("Привет")                    # одна строка
print("Мне", age, "лет")           # запятая → пробел между частями
print("Я из " + city + ".")        # склейка строк через +
```

| Способ | Пробелы |
|--------|---------|
| `print(a, b, c)` | автоматически между аргументами |
| `print(a + b)` | только те, что ты сам вставил в строки |

## Строки

```python
"текст"           # двойные кавычки
'текст'           # одинарные — то же самое
name + "!"        # склейка: "Сергей" + "!" → "Сергей!"
```

**Важно:** `"10" + "3"` → `"103"` (склейка), а `10 + 3` → `13` (сложение).

## `int()` — строка → целое число

```python
age = int(input("Сколько лет? "))
a = int(input("Первое число: "))
```

Цепочка для арифметики с вводом: `input` → строка → `int()` → число.

## Арифметика

| Оператор | Действие | Пример |
|----------|----------|--------|
| `+` | сложение | `10 + 3` → `13` |
| `-` | вычитание | `10 - 3` → `7` |
| `*` | умножение | `10 * 3` → `30` |
| `/` | деление | `10 / 3` → `3.333...` (может быть дробь) |

```python
summ = a + b
avg = (x + y) / 2
volume = x * y * z
```

## Сравнения (для `if`)

| Оператор | Значение |
|----------|----------|
| `==` | равно |
| `!=` | не равно |
| `>` | больше |
| `<` | меньше |

**Не путать:** `=` — присваивание, `==` — сравнение.

## `if` / `else` — ветвление

```python
if age >= 18:
    print("Взрослый")
else:
    print("Ребёнок")
```

```python
if number_1 > number_2:
    print("Первое больше")
if number_1 == number_2:
    print("Числа равны")
if number_1 < number_2:
    print("Первое меньше")
```

Правила:
- после `if` / `else` — **двоеточие** `:`
- код внутри блока — **с отступом** (обычно 4 пробела)
- без отступа код **вне** блока

## `if` / `elif` / `else` — выбор одного варианта (день 10)

```python
if op == "+":
    print(a + b)
elif op == "-":
    print(a - b)
elif op == "*":
    print(a * b)
elif op == "/":
    print(a / b)
else:
    print("Неизвестная операция")
```

`elif` — «иначе если»; срабатывает **первая** подходящая ветка.

**`and`** — оба условия истинны:

```python
if op == "/" and b == 0:
    print("На ноль делить нельзя!")
```

**`or`** — хотя бы одно условие истинно:

```python
while age < 1 or age > 120:
    print("Нужен возраст от 1 до 120")
    age = int(input("Возраст: "))
```

**`not`** — отрицание (True ↔ False):

```python
if not is_closed:
    print("Открыто")
```

## Цикл `for` (день 8)

```python
for i in range(5):
    print(i)          # 0, 1, 2, 3, 4
```

```python
for n in range(1, 4):
    print(n)          # 1, 2, 3
```

Синтаксис:

```python
for переменная in последовательность:
    тело цикла       # с отступом — выполняется на каждом шаге
print("Готово")      # без отступа — один раз, после цикла
```

---

## `range()` — набор чисел для цикла

| Вызов | Что перебирается |
|-------|------------------|
| `range(5)` | 0, 1, 2, 3, 4 |
| `range(1, 6)` | 1, 2, 3, 4, 5 |
| `range(1, 11)` | 1 … 10 |
| `range(0, 10, 2)` | 0, 2, 4, 6, 8 |

**Правило:** `range(начало, конец)` — **конец не входит**.  
Чтобы дойти до 10 включительно: `range(1, 11)`.

## Цикл `while` (день 11)

```python
while условие:
    тело цикла    # с отступом; внутри — менять то, от чего зависит условие
```

Проверка **перед каждым** шагом. Зависание → **Ctrl+C**.

| | `for` | `while` |
|---|--------|---------|
| Когда | шагов известно заранее | «пока условие» |

Идея повторного ввода: спросить → `while` «не подходит» → спросить снова. Конкретное условие — из задачи.

## `_` — когда номер шага не нужен

```python
    print("Привет")   # напечатает 5 раз, номер шага не используем
```

## Функции `def` (день 13)

```python
def area(w, h):
    return w * h

s = area(5, 10)   # вызов — результат в переменной
```

```python
def greet(name):
    print("Привет,", name)
```

| | |
|---|---|
| `def` | определить функцию (обычно **выше** вызова) |
| параметры | данные в `()` |
| `return` | отдать значение |
| `print` в функции | только показать, не вернуть |

## Строки: методы (день 14)

```python
len("hello")           # 5 — длина
text.strip()           # без пробелов по краям
text.lower()           # маленькие буквы
text.upper()           # большие буквы
```

```python
answer = input("да/нет: ").strip().lower()
if answer == "да":
    ...
```

Метод **возвращает новую** строку — часто нужно: `text = text.lower()`

```python
for ch in "abc":
    print(ch)          # перебор символов
```

## Частые шаблоны

**Диалог с пользователем:**

```python
name = input("Имя: ")
print("Привет,", name)
```

**Два числа и сумма:**

```python
a = int(input("Число 1: "))
b = int(input("Число 2: "))
print("Сумма:", a + b)
```

**Проверка условия:**

```python
temp = int(input("Температура: "))
if temp <= 0:
    print("Холодно!")
else:
    print("Тепло!")
```

**Числа от 1 до 10:**

```python
for i in range(1, 11):
    print(i)
```

**Сумма от 1 до N (идея):**

```python
n = int(input("N: "))
total = 0
for i in range(1, n + 1):
    total = total + i
print("Сумма:", total)
```

## Быстрые напоминания об ошибках

| Ошибка | Что помнить |
|--------|-------------|
| `TypeError` при `+` | смешал строку и число — нужен `int()` |
| `"12" + "3"` → `"123"` | это склейка, не сложение |
| `range(1, 10)` без десятки | конец не входит → нужно `range(1, 11)` |
| `IndentationError` | забыл отступ после `if` / `for` / `else` |
| код не в цикле | `print` без отступа выполнится один раз |
| произведение всегда 0 | старт должен быть `1`, не `0` |
| `print` вместо `return` | функция ничего не «отдаёт» в переменную |

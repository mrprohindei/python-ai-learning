## Как работать

1. Открой `day_XXD.py` — вверху указаны **темы** и файлы теории.
2. Пиши код в этом же файле.
3. Запуск:

   ```powershell
   python problem_D/day_11D.py
   ```

4. Проверка в чате: «Проверь day_11D» + `@problem_D/day_11D.py`

Закрытие урока **не требует** D — только A и B в `lessons/day_XX.py`.
